# Real Estate KNN Valuation Tool

## Objective
This project aims to estimate market rents using comparable properties selected with a K-Nearest Neighbors (KNN) approach.

## How it works
- A project is defined with its characteristics
- The model retrieves the nearest comparable assets
- An estimated rent is computed from the selected comparables

## Methodology
### Comparable selection
The tool identifies the closest properties based on relevant features such as location, size, and building characteristics.

### Rent estimation
Two approaches can be used:
- Simple average of comparable rents
- Distance-weighted average, giving more weight to the closest assets

## Tech stack
- SQL Server
- Python
- Power BI

## Output
- Comparable assets
- Distances
- Estimated rent
- Transparent valuation logic
